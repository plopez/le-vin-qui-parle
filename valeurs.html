SIMPLICITE

Nous avons un vrai recul sur l'agilité pour avoir vécu plusieurs dizaines de projets agiles que ce soit en Extreme Programming, Scrum ou Kanban.

Nous pratiquons l'agilité aussi bien à l'échelle de l'organisation du projet que sur les pratiques techniques.

Nous développons les fonctions qui ont de la valeur pour vous et vos utilisateurs, nous contribuons à votre reflexion sur le fonctionnel de votre application.

EXCELLENCE

Notre motivation pour exceller se résume par la phrase : "Tolérer la médiocrité c'est l'encourager." Nous combattons le code médiocre, les applications médiocres, les processus médiocres.
